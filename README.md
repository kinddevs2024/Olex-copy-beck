# Olex-copy-beck

A minimal backend API scaffold for the Olex-copy React Native frontend.

## What is included

- `server.js`: Express server with CORS enabled.
- `routes/cars.js`: sample car endpoints for listing, retrieving, and creating listings.
- `package.json`: Node dependencies and start script.

## Run locally

1. Open `Olex-copy-beck`.
2. Run `npm install`.
3. Run `npm start`.
4. The API will be available at `http://localhost:4000`.

## Available endpoints

- `GET /api/cars`
- `GET /api/cars/:id`
- `POST /api/cars`
